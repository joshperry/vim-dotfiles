VIM Dotfiles
============

## New System Setup

Uses [Vundle](/gmarik/vundle) for package management, 

1. Install Vundler
     ```
     $ git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
     ```

2. Install Packages
	`vim +BundleInstall +qall`

3. Link .vimrc
	`ln -s ~/.vim/config-vimrc ~/.vimrc
